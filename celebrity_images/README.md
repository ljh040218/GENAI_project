# 📸 celebrity_images 폴더 설명

이 폴더에는 **연예인별 메이크업 분석용 이미지 데이터**가 저장되어 있습니다.  
모델 학습 및 테스트를 위해 얼굴 부위별 색상, 질감, 조명 분석을 수행할 때 활용됩니다.

---

## 📂 폴더 구조
| 폴더명 | 설명 |
|--------|------|
| `jennie/` | 제니 이미지 샘플 |
| `wonyoung/` | 장원영 이미지 샘플 |
| `karina/` | 카리나 이미지 샘플 |

> 각 폴더는 동일한 구조로 되어 있으며, 인물별로 약 40~50장의 이미지가 포함되어 있습니다.

---

## 💄 활용 목적
- **얼굴 부위 분할(face parsing)**  
  → SegFormer 기반 segmentation 모델 학습  
- **색상 추출 및 tone 분석**  
  → LAB 색공간 변환을 통한 피부톤 및 화장품 색조 분석  
- **유사 룩 추천**  
  → 사용자의 얼굴 사진과 색상 유사도를 계산하여 추천 제품 탐색

---

## ⚠️ 주의 사항
- 이 데이터는 연구 및 개발 목적의 **비상업적 용도**로만 사용됩니다.  
- 초상권 및 저작권 보호를 위해 외부 배포 금지.  
- 필요 시 개인 식별이 불가능하도록 얼굴 마스킹/크롭 처리 후 사용해야 합니다.
