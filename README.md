---

## 📌 프로젝트 소개

**VIZY**는 사용자 얼굴 이미지와 화장품 이미지를 분석하여  
개인 톤에 맞는 K-Beauty 제품을 추천하는 AI 기반 뷰티 추천 서비스입니다.

조명, 카메라 환경, 필터 등에 의해 달라 보이는 제품 색상을  
딥러닝 기반 얼굴 분석 + 색상 거리 계산을 통해  
보다 **정확하고 직관적인 추천**으로 제공합니다.

## 🚀 주요 기능

### ✅ 얼굴 기반 AI 추천
- 얼굴 이미지 업로드
- Face Parsing을 통한 피부 영역 분석
- Lab 색 공간 변환 및 톤 분석

### ✅ 제품 이미지 기반 추천
- 화장품 사진 업로드
- 주요 색상 추출
- 기존 데이터베이스 제품과 색상 비교

### ✅ 컬러 유사도 기반 매칭
- ΔE (Delta E) 거리 계산 기반 추천
- 사용자 톤과 제품 색상 통합 점수 산정

### ✅ AI 추천 설명
- LLM 기반 추천 사유 자동 생성
- "왜 이 제품이 나에게 맞는지" 설명 제공
